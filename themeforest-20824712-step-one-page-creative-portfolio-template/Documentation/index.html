What is this?

STEP is an OnePage HTML5 Template built with HTML5 and CSS3 that have well-organized with valid code. This documentation provide you a clear & full guideline for how to use this OnePage Template.

The template is based on bootstrap library, if you donâ€™t know about bootstrap please take a look at their website http://getbootstrap.com/getting-started/. Actually in this template you will not have to know a lot in bootstrap, just the basic things.

You can edit the Html file in any text editor, but we prefer to use an html editor like Adobe Dreamweaver.

HTML Structure

Logo

To change the logo, replace the img src=" " with your logo url.

Change the Red marked values from index page 
    <div class="logo">
        <!-- change logo here -->
        <a href="#home"><img src="img/logo.png" alt="step"></a>
    </div>

Menu

To change the menu links, simply edit this code. For example, the contact link is linked to the Contact section.

    <nav class="text-center text-uppercase">
       .....
        <ul>
            <!-- Menu Items -->
            <li><a href="#home">Home</a></li>
            <li><a href="#about">About</a></li>
            <li><a href="#portfolio">Portfolio</a></li>
            <li><a href="#service">Service</a></li>
            <li><a href="#testimonial">Testimonial</a></li>
            <li><a href="#blog">Blogs</a></li>
            <li><a href="#contact">Contact</a></li>
        </ul>
    </nav><!-- end nav menu -->

About Section

To change the about section avater photo, replace the img src=" " with your avater url.

Change the Red marked values from index page 
    <img class="img-responsive img-circle" src="img/person.jpg" alt="Mr. John">

To change the Skill value edit the inline-style width from class="skillBar" and change the numericTEXT from class="skillText" same as class="skillBar" width.

Change the Red marked values from index page 
    <div class="skillMain">
        <div class="skillBar wow slideInLeft" style="width:92%">
            <span class="skillText wow zoomIn" data-wow-delay=".6s">Web - 92%</span>
        </div>
    </div>

To change the Counter value edit the numericTEXT from class="counter" and change the name from span tag

Change the Red marked values from index page 
    <section class="expreince_wrapper wow fadeIn">
        .....

                <i class="icofont icofont-users-social"></i>

    </section>

Changing Font

By default, STEP uses 1 Font namely: 'Nunito' from the Google Fonts Library. To change the Font, you will need to Edit the Bottom Link with your Custom Font

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

        @import url('https://fonts.googleapis.com/css?family=Nunito:300,400,700');
